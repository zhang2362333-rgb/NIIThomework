# Spring Setter Dependency Injection Demo

This project demonstrates how to implement Setter-based dependency injection using Spring XML configuration. 
Through the `applicationContext.xml` configuration file, basic type values (name) and object type values (Address) 
are injected into the Student Bean without using the `new` keyword in the code.

## Core Features
- ✅ Manage beans using XML configuration
- ✅ Demonstrate Setter injection (`value=` for values, `ref=` for objects)
- ✅ Showcase Spring's Inversion of Control (IoC) concept
